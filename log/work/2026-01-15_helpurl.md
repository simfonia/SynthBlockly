# 2026-01-15 工作日誌 (SynthBlockly) - HelpUrl Refactoring

## 異動摘要
- **新增工具函式**：建立 `js/core/helpUtils.js`，集中管理 HelpUrl 的生成邏輯，自動處理 Vite 的 `BASE_URL` 與語系切換 (`_zh-hant` / `_en`)。
- **全域重構**：將 `js/blocks/` 下所有積木定義檔中的 `setHelpUrl` 實作替換為 `getHelpUrl('filename_base')`。

## 技術細節
- `getHelpUrl` 回傳一個閉包函式，確保 Blockly 在需要時才計算 URL，從而支援動態語系切換而無需刷新頁面。
- 使用 `import.meta.env.BASE_URL` 確保在 GitHub Pages 或其他子目錄部署環境下路徑正確。
