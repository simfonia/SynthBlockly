# 電子鼓墊 (單鍵模式)

此範例展示如何將 TTP229 觸控板的按鍵訊號轉換為鼓聲與和弦。

### 1. 功能說明
- **即時響應**：利用「帽子積木」監聽序列埠，無需點擊「執行」即可發聲。
- **單鍵觸發**：監聽來自 Arduino 的字串訊息（例如 `KEY:1`）。
- **多音色整合**：同時使用合成鼓組 (JazzKit) 與鋼琴和弦。

### 2. 硬體需求
- Arduino 開發板 x1
- TTP229 觸控板 x1
- 接線說明：請參考本目錄下的 `.ino` 檔案註解。

### 3. 操作對照
- **按鍵 1**：觸發大鼓 (Kick)。
- **按鍵 2**：觸發拍手聲 (Handclap)。
- **按鍵 3**：演奏 C Major 和弦。
- **按鍵 4**：演奏 F Major 和弦。

### 4. 軟體設定
1. 點擊「下載 Arduino 程式碼 (.ino)」。
2. 使用 Arduino IDE 燒錄程式至開發板。
3. 回到網頁，點擊工具列的「連接序列埠」。
4. 開始敲擊觸控板！
