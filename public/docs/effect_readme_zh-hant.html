效果器積木詳細指南

通用參數

  • 濕度 (Wet): 控制效果的乾濕混合比例 (0 到 1)。數值為 0 表示聽不到效果 (乾聲)，數值為 1 表示只聽到效果聲 (濕聲)。

失真 (Distortion)

為聲音增添諧波、砂礫感和飽和度。

  • 量 (Amount): 失真的強度，範圍從 0 (乾淨) 到 1 (嚴重失真)。
  • 超取樣 (Oversample): 減少因數位取樣產生的高頻混疊雜訊。使用 '2x' 或 '4x' 可以獲得更平滑、更高品質的聲音，但會稍微增加處理器負擔。

混響 (Reverb)

模擬聲音在特定空間（如房間、大廳等）中的反射效果。

  • 衰減 (Decay): 混響尾音的持續時間（秒）。較長的衰減時間模擬較大的空間。
  • 預延遲 (Predelay): 原始聲音與第一個混響反射音之間的時間差（秒）。短暫的預延遲有助於保持原始聲音的清晰度。

回饋延遲 (Feedback Delay)

產生輸入聲音的回音。

  • 延遲時間 (Delay Time): 連續回音之間的間隔時間。可以是音樂時值（如 '8n' 代表八分音符）或以秒為單位的時間。
  • 回饋 (Feedback): 回音的強度，範圍從 0 到 1。數值 0.25 表示每個回音的音量是前一個的 25%。過高的數值可能導致自我振盪。

濾波器 (Filter)

透過增強或削減特定頻率來塑造聲音的音色特徵。

  • 類型 (Type): 應用的濾波器種類（例如 lowpass 讓低頻通過，highpass 讓高頻通過）。
  • 頻率 (Frequency): 濾波器的截止或中心頻率，單位為赫茲 (Hz)。
  • Q (共振): 濾波器的頻寬。對於低通/高通濾波器，較高的 Q 值會在截止頻率處產生一個共振峰；對於帶通濾波器，則會使頻帶變窄。
  • 滾降斜率 (Rolloff): 濾波器截止的陡峭程度，單位為分貝/八度。較陡峭的數值（如 -24, -48）會產生更強烈的濾波效果。

壓縮器 (Compressor)

減小聲音的動態範圍，使小聲的部分變大聲，大聲的部分變小聲。

  • 閾值 (Threshold): 開始壓縮的音量水平，單位為分貝 (dB)。範圍約為 -100 到 0。較低的數值（如 -50）意味著更多信號會被壓縮。
  • 比率 (Ratio): 施加的壓縮量。例如，比率為 4 表示輸入信號每超過閾值 4 dB，輸出信號只會增加 1 dB。
  • 起始時間 (Attack): 當信號超過閾值後，壓縮器開始降低增益所需的時間（秒）。
  • 釋放時間 (Release): 當信號低於閾值後，壓縮器停止降低增益所需的時間（秒）。

限幅器 (Limiter)

一種極端的壓縮器，可防止信號超過設定的閾值。

  • 閾值 (Threshold): 信號不允許超過的音量水平，單位為分貝 (dB)。最大值為 0。範圍約為 -100 到 0。
