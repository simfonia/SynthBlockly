音訊效果器：原理與參數詳解

透過「設定效果器」積木，你可以為樂器添加各種專業的後製效果。了解每個參數的意義與範圍，能幫助你精確掌控聲音的細節。

通用參數：乾濕比 (Wet)

這代表了「原始訊號 (Dry)」與「處理後訊號 (Wet)」的混合比例。0 為純原音，1 為純效果音。通常設為 0.5 能在保留原音清晰度的同時獲得明顯的效果。

1. 混響 (Reverb)

物理原理：模擬聲波在空間內多次反射形成的餘韻（尾巴）。
訊號處理：利用延遲矩陣模擬空間大小與牆面吸收率。
調整參數：
  • 衰減 (Decay) [0.1 - 10+ 秒]：數值越大，空間聽起來越開闊（如大教堂）；越小則像小房間。
  • 預延遲 (PreDelay) [0 - 0.1 秒]：原始聲音與第一次反射聲之間的時間差，增加此值可讓聲音更具層次感。
  • 濕度 (Wet) [0 - 1]：效果音的比例。

2. 回饋延遲 (Feedback Delay)

物理原理：模擬聲波遇到障礙物折返的回音現象。
訊號處理：將訊號暫存後播放，並將部分輸出送回輸入端產生重複。
調整參數：
  • 延遲時間 (Delay Time) [音樂時值，如 4n, 8n]：回音出現的時間間隔。
  • 回饋 (Feedback) [0 - 1]：決定回音重複的次數。0 為一次，接近 1 則會持續很久。
  • 濕度 (Wet) [0 - 1]：原音與回音的混合比。

3. 濾波器 (Filter)

物理原理：模擬材質對不同頻率聲波的阻擋效果。
訊號處理：選擇性地通過或移除特定頻率範圍。
濾波器類型詳解：
  • lowpass (低通)：只允許低於頻率的聲音通過。會移除高頻，使聲音變得悶悶的（像隔著牆聽音樂）。
  • highpass (高通)：只允許高於頻率的聲音通過。會移除低頻，使聲音變得尖銳、單薄。
  • bandpass (帶通)：只允許頻率附近的聲音通過。會同時移除極高與極低的聲音，產生類似電話或收音機的效果。
  • lowshelf (低架)：調整低頻段的音量增益，而不完全切除。常用於增強或減弱重低音。
  • highshelf (高架)：調整高頻段的音量增益。常用於調整聲音的明亮度。
  • notch (陷波)：極度精準地移除特定頻率附近的聲音。常用於消除特定的雜音或回授。
  • allpass (全通)：讓所有頻率通過，但改變它們的相位關係。常用於改變音色的質感。
  • peaking (峰值)：增強或減弱特定頻率附近的聲音。類似一般等化器 (EQ) 的調整。
調整參數：
  • 頻率 (Frequency) [20 - 20000 Hz]：效果作用的基準點。
  • Q值 (Resonance) [0.1 - 20]：共振強度。數值高會在切斷點產生強調感，產生類似口哨或變頻感。
  • 滾降 (Rolloff) [-12, -24, -48]：衰減斜率。數字越大表示切除得越乾淨、越生硬。

4. 失真 (Distortion)

物理原理：模擬電路過載導致波形被削平產生的破音。
訊號處理：非線性放大訊號，產生豐富的諧望（奇偶次諧波）。
調整參數：
  • 量 (Amount) [0 - 1]：失真的程度。0 為微弱飽和，1 為極度狂野的噴破聲。
  • 超取樣 (Oversample)：改善高頻失真的品質，減少數位噪音。
  • 濕度 (Wet) [0 - 1]：原音與失真音的比例。

5. 壓縮器 (Compressor) 與 限幅器 (Limiter)

物理原理：自動化控制音量，讓聲音更穩定或防止爆音。
訊號處理：當音量超過閾值時，依照比率將音量壓低。
調整參數：
  • 閾值 (Threshold) [-100 - 0 dB]：超過此音量後開始作用。
  • 比率 (Ratio) [1 - 20]：壓縮的強度。Limiter 固定為極高比率。
  • 起始/釋放時間：壓縮作用與消失的速度。

6. 調變類效果 (Chorus / Phaser / Tremolo / AutoPanner)

物理原理：透過時間、相位或音量的週期性變化產生流動感。
  • 合唱 (Chorus)：模擬多個音源同時發聲。
  • 移相 (Phaser)：產生類似噴射機掠過的旋轉感。
  • 顫音 (Tremolo)：快速改變音量產生波動感（如電吉他）。
  • 自動相位 (AutoPanner)：讓聲音在左右聲道間來回移動。
共同參數：
  • 頻率 (Frequency) [Hz]：效果變化的週期速度。數值越高，波動越快。
  • 深度 (Depth) [0 - 1]：效果作用的強度（如音量起伏大小或左右移動範圍）。
  • 濕度 (Wet) [0 - 1]：效果音的混合比。

7. 復古效果 (Lo-Fi)

調整參數：
  • 位元深度 (Lofi Bits) [4 - 16]：數值越低，聲音越具有粗糙的 8-bit 復古電玩感。
  • 濕度 (Wet) [0 - 1]：效果音的混合比。
