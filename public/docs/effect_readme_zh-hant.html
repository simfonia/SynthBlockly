效果器積木詳細指南

通用參數

  • 濕度 (Wet): 控制效果的乾濕混合比例 (0 到 1)。數值為 0 表示聽不到效果 (乾聲)，數值為 1 表示只聽到效果聲 (濕聲)。

失真 (Distortion)

「失真」是透過非線性方式對音訊訊號進行處理。它會刻意改變輸入波形的形狀，通常是將其峰值「削平」或「飽和化」。這種非線性操作會引入原始訊號中不存在的新的諧波成分（泛音），特別是奇次諧波，從而產生「毛躁」或「粗糙」的聽感。

為聲音增添諧波、砂礫感和飽和度。

  • 量 (Amount): 失真的強度，範圍從 0 (乾淨) 到 1 (嚴重失真)。
  • 超取樣 (Oversample): 減少因數位取樣產生的高頻混疊雜訊。使用 '2x' 或 '4x' 可以獲得更平滑、更高品質的聲音，但會稍微增加處理器負擔。

混響 (Reverb)

「混響」模擬了聲音在封閉空間中反射的物理現象。當聲音在房間中產生時，它會撞擊牆壁、天花板、地板等表面，產生無數次的反射。這些反射以微小的時間差和逐漸衰減的響度到達聽者耳中。混響效果正是透過生成一系列密集且逐漸衰減的「回音」，讓這些回音相互融合，從而營造出空間感。

模擬聲音在特定空間（如房間、大廳等）中的反射效果。

  • 衰減 (Decay): 混響尾音的持續時間（秒）。較長的衰減時間模擬較大的空間。
  • 預延遲 (Predelay): 原始聲音與第一個混響反射音之間的時間差（秒）。短暫的預延遲有助於保持原始聲音的清晰度。

回饋延遲 (Feedback Delay)

「回饋延遲」透過將一部分延遲後的訊號再次送回延遲線的輸入端，來產生一系列逐漸衰減的回音。原始訊號首先被延遲，然後一小部分的延遲訊號被重新加回到輸入，使其再次被延遲，如此循環，便產生了連續且音量逐漸減小的重複聲音。

產生輸入聲音的回音。

  • 延遲時間 (Delay Time): 連續回音之間的間隔時間。可以是音樂時值（如 '8n' 代表八分音符）或以秒為單位的時間。
  • 回饋 (Feedback): 回音的強度，範圍從 0 到 1。數值 0.25 表示每個回音的音量是前一個的 25%。過高的數值可能導致自我振盪。

濾波器 (Filter)

「濾波器」透過衰減或增強特定的頻率範圍，來改變聲音的頻譜內容。它的原理是選擇性地讓某些頻率通過，同時減少其他頻率。這種操作改變了聲音的音色，因為它重新塑造了聲音的頻譜平衡。

透過增強或削減特定頻率來塑造聲音的音色特徵。

  • 類型 (Type): 應用的濾波器種類（例如 lowpass 讓低頻通過，highpass 讓高頻通過）。
  • 頻率 (Frequency): 濾波器的截止或中心頻率，單位為赫茲 (Hz)。
  • Q (共振): 濾波器的頻寬。對於低通/高通濾波器，較高的 Q 值會在截止頻率處產生一個共振峰；對於帶通濾波器，則會使頻帶變窄。
  • 滾降斜率 (Rolloff): 濾波器截止的陡峭程度，單位為分貝/八度。較陡峭的數值（如 -24, -48）會產生更強烈的濾波效果。

壓縮器 (Compressor)

「壓縮器」旨在減小音訊訊號的動態範圍。當訊號的響度（振幅）超過設定的「閾值」時，壓縮器會依照一個特定的「比率」自動降低其增益（音量）。這使得響亮的部分相對於原始訊號變得較安靜，從而有效縮小了聲音最響亮和最安靜部分之間的差異。

減小聲音的動態範圍，使小聲的部分變大聲，大聲的部分變小聲。

  • 閾值 (Threshold): 開始壓縮的音量水平，單位為分貝 (dB)。範圍約為 -100 到 0。較低的數值（如 -50）意味著更多信號會被壓縮。
  • 比率 (Ratio): 施加的壓縮量。例如，比率為 4 表示輸入信號每超過閾值 4 dB，輸出信號只會增加 1 dB。
  • 起始時間 (Attack): 當信號超過閾值後，壓縮器開始降低增益所需的時間（秒）。
  • 釋放時間 (Release): 當信號低於閾值後，壓縮器停止降低增益所需的時間（秒）。

限幅器 (Limiter)

「限幅器」是一種具有極高（通常是無限大）壓縮比的特殊壓縮器，旨在防止音訊訊號的響度超過絕對的最大值（閾值）。任何試圖超過此閾值的訊號部分都會立即被壓低到閾值水平，從而有效防止了削波失真和不必要的音量峰值。

一種極端的壓縮器，可防止信號超過設定的閾值。

  • 閾值 (Threshold): 信號不允許超過的音量水平，單位為分貝 (dB)。最大值為 0。範圍約為 -100 到 0。

低傳真 (Lo-Fi)

「低傳真 (Lo-Fi)」效果透過刻意降低音訊品質來創造其獨特的聲音，主要手段包括位元破壞 (Bit Crushing) 和/或降採樣 (Downsampling)。

  • 位元破壞 (Bit Crushing): 減少訊號的數位位元深度。這意味著用來表示波形的離散振幅層次變少，會引入量化誤差和雜訊，導致波形產生「階梯狀」變化，形成顆粒感和數位失真。
  • 降採樣 (Downsampling): 降低訊號的採樣率。這會限制能被精確表示的最高頻率（奈奎斯特頻率），並引入混疊 (aliasing) 現象（高頻率會折疊回較低頻率），從而產生較不細膩且帶有「低傳真」感的聲音。

模擬早期數位設備的音質，透過降低位元深度來產生獨特的顆粒感和復古效果。

  • 位元深度 (Bit Depth): 設定音訊的位元深度。較低的位元深度會產生更明顯的 Lo-Fi 效果和量化雜訊。可選值為 4, 8, 12, 16。

合唱 (Chorus)

「合唱」效果旨在模擬多個聲部或樂器稍微不同步演奏的聽感。其原理是複製原始訊號一次或多次，然後對每個複製品進行微小的延遲，並使用一個緩慢的低頻振盪器 (LFO) 持續調變這些延遲時間。這種動態延遲會造成聲音輕微且持續的音高變化（都卜勒效應），從而產生飽滿、閃爍且寬廣的音場效果。

透過將原始聲音與其稍微延遲且音高略有變化的複製品混合，來模擬多個樂器同時演奏的效果，使聲音聽起來更飽滿、更寬廣。

  • 頻率 (Frequency): LFO（低頻振盪器）的頻率，單位為赫茲 (Hz)。這個頻率控制著音高調變的速度。
  • 延遲時間 (Delay Time): 原始聲音與複製品之間的基礎延遲時間，單位為毫秒 (ms)。
  • 深度 (Depth): 調變的深度，範圍從 0 到 1。這個值決定了音高變化的幅度。

相位器 (Phaser)

相位器利用聲音的相位偏移 (Phase Shift) 和波的干涉 (Wave Interference) 來產生其獨特效果。它會將聲音訊號分成兩路：一路保持原始狀態，另一路則通過一系列特殊濾波器，這些濾波器會根據頻率不同地改變聲音的相位，但不改變響度。當這兩路訊號重新混合時，在某些頻率點會因相位同步而相互增強，而在另一些頻率點則因相位錯開而相互抵消，形成獨特的「梳狀濾波 (Comb Filtering)」效果。

為了產生標誌性的「掃頻」聲，相位器會使用一個低頻振盪器 (LFO) 緩慢且連續地改變這些濾波器的設定，使得梳狀濾波的「齒狀凹陷」在頻譜上來回移動，從而產生旋渦或呼嘯般的聽感。

相位器透過創建一系列交錯的濾波器凹口 (notches)，為聲音帶來獨特的掃頻和旋轉效果。

  • 頻率 (Frequency): LFO（低頻振盪器）的頻率，單位為赫茲 (Hz)。控制相位掃描的速度。
  • 八度音程 (Octaves): 相位掃描的頻率範圍，以八度音程表示。較高的值會導致更寬廣的頻率掃描。
  • 基礎頻率 (Base Frequency): 相位掃描的起始頻率，單位為赫茲 (Hz)。

自動搖攝 (AutoPanner)

「自動搖攝」的核心原理是透過振幅調變 (Amplitude Modulation) 來模擬聲音在立體聲場中的移動。它使用一個低頻振盪器 (LFO) 來連續且獨立地改變左右聲道的音量（振幅）。通常，左聲道的音量會隨著 LFO 波形的升高而增加，同時右聲道的音量會隨著 LFO 波形的反向（反相）而降低。當左右聲道的相對音量動態變化時，人腦會感知到聲源在立體聲圖像中水平移動，從而營造出聲源在空間中「游移」的效果。

「自動搖攝」效果會透過一個低頻振盪器 (LFO) 自動改變聲音在立體聲場中的位置，使聲音在左右聲道之間來回移動。這模擬了聲音從一個方向移動到另一個方向的效果，增加了聲音的空間感和動態性。

  • 頻率 (Frequency): LFO 的頻率，單位為赫茲 (Hz)。控制聲音在立體聲場中移動的速度。
  • 深度 (Depth): LFO 的振幅，範圍從 0 (不移動) 到 1 (在左右聲道之間完全移動)。控制聲音移動的幅度。
