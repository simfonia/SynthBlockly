3. Workflow for Hardware Examples

  1. After loading, read the "Hardware Setup Guide" window carefully.
  2. Click "Download Arduino Code (.ino)" to get the corresponding C++ program.
  3. Open the downloaded file in the Arduino IDE and upload it to your board.
  4. Complete the circuit wiring according to the instructions (e.g., buttons, potentiometers, or sensors).
  5. Return to SynthBlockly and click "Connect Serial" to link with your Arduino.

Tip: Feel free to modify the block parameters to experiment with different sound effects!

4. How to Add Custom Examples (For Developers)

If you want to add your own examples, follow these conventions in the src/examples/ directory. The system scans files automatically based on extensions:

  • Software Example: Simply add an .xml file directly to the folder (e.g., src/examples/MySong.xml).
  • Hardware Example (Folder-based): Create a folder (e.g., src/examples/MyProject/). The system will match files as follows:
    • Blockly Entry: The filename must be main.xml or match the folder name (e.g., MyProject.xml).
    • Arduino Code: Any .ino file will be detected. The filename is flexible and can be placed anywhere within the folder.
    • Documentation: Any .md file (e.g., README.md) will be detected. The filename is flexible, and its content will be displayed in the hardware guidance window.
