合成器型態說明

在「創建樂器」積木中，你可以選擇以下四種不同的合成技術來產生聲音：

1. PolySynth (標準合成器)

原理：最基礎的減法合成技術（Subtractive Synthesis）。

特性：音色純淨、直觀，是合成器世界的「白開水」。

適用：基礎電子琴、Lead 音色、簡單的 Bass。非常適合初學者理解 ADSR 的變化。

2. AMSynth (振幅調變合成)

原理：使用一個振盪器去快速控制另一個振盪器的音量 (Amplitude)。

特性：帶有一種獨特的「顫動感」，聽起來像是在快速震動的風琴聲。

適用：復古電子音效、特殊風琴音色。

3. FMSynth (頻率調變合成)

原理：使用一個振盪器去快速控制另一個振盪器的音高 (Frequency)。

特性：音色複雜、明亮，帶有強烈的數位感與金屬感。

適用：鈴聲 (Bells)、金屬打擊音色、銳利的數位 Bass。這是 80 年代流行音樂的經典聲音。

4. DuoSynth (雙重合成器)

原理：疊加兩個獨立的合成器聲音，形成厚實的音層。

特性：音色非常渾厚、強大且富有層次。

適用：宏偉的合成器墊音 (Pads)、強力的銅管 (Brass) 或極具侵略性的 Lead 音色。

減法合成 vs 加法合成

這兩種合成方式是聲音設計中最基礎也最對立的兩大門派，猶如雕刻與堆疊的差異。

1. 減法合成 (Subtractive Synthesis) - 像「雕刻」

這是最常見的合成方式，例如 PolySynth。

  • 核心邏輯：由多變少。先產生豐富的波形（如鋸齒波），再用濾波器切掉不需要的頻率。
  • 現實比喻：在一塊大理石上，鑿掉不要的部分，雕出雕像。
  • 聽感：溫暖、有力、經典。適合 Bass、Lead 和 Pad。

2. 加法合成 (Additive Synthesis) - 像「堆積木」

例如 自訂波形樂器 (諧波/自由頻率)。

  • 核心邏輯：由少變多。從「無」開始，將多個純淨的正弦波 (Sine Wave) 疊加起來。
  • 現實比喻：像用樂高積木一塊塊堆出形狀，或點描畫派的畫作。
  • 聽感：明亮、金屬感、玻璃感。適合鐘聲 (Bell)、風琴 (Organ) 或科幻音效。
