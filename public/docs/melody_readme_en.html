Melody List Writing Guide

The "Play Melody List" block uses a simplified text format to represent Tone.js musical durations. This allows you to code complex rhythms quickly.

1. Duration Codes Table

This block uses single letters as shorthands, mapping to standard Tone.js notation as follows:

Code Name Tone.js Syntax Description
W Whole Note 1m / 1n Lasts for a full measure (4 beats).
H Half Note 2n Lasts for 2 beats.
Q Quarter Note 4n Lasts for 1 beat.
E Eighth Note 8n Lasts for 0.5 beats.
S Sixteenth Note 16n Lasts for 0.25 beats.
T 32nd Note 32n Very short note.

2. Rests (Silence)

Silence is music too. Use R followed by a duration code to create a rest.

  • RQ: Quarter Rest (Wait 1 beat).
  • RH: Half Rest (Wait 2 beats).
  • RE: Eighth Rest.

3. Advanced Rhythms (Modifiers)

You can combine modifiers to create richer grooves:

Dotted Note - .

Adding a . after the duration increases the length by 50% (1.5x).

  • Q. (Dotted Quarter) = 1 beat + 0.5 beat = 1.5 beats.
  • H. (Dotted Half) = 2 beats + 1 beat = 3 beats.
  • Tone.js Equivalent: 4n., 2n.

Triplets - _T

Adding _T after the duration means playing 3 notes in the time of 2 (Length is 2/3 of original).

  • Q_T (Quarter Triplet): Length is 2/3 of a standard quarter note.
  • E_T (Eighth Triplet): Common in Swing or Waltz music.
  • Tone.js Equivalent: 4t, 8t

4. Full Format & Examples

Single Note Format: [Pitch][Octave][Duration][Modifier]

Example 1 (Basic): C4Q E4Q G4H (Do 1 beat, Mi 1 beat, Sol 2 beats) Example 2 (With Rests): C4Q RQ E4Q RQ (Do 1 beat, Rest 1 beat, Mi 1 beat, Rest 1 beat) Example 3 (Complex): C4Q. D4E E4Q_T E4Q_T E4Q_T (Dotted Quarter, Eighth, three Triplets)
